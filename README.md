# 📘 Curso Java — Exceções

Repositório criado para armazenar os códigos, exemplos e práticas desenvolvidas durante o curso **Java exceções: aprenda a criar, lançar e controlar exceções**, da **Alura**.

Este projeto tem como objetivo consolidar os conceitos fundamentais de **tratamento de exceções em Java**, com foco em controle de erros, organização do código e boas práticas no desenvolvimento de aplicações, incluindo o uso em APIs com Spring Boot.

---

## 🧠 Conteúdos abordados

Durante o curso, foram estudados os seguintes tópicos:

- Conceito de **exceções em Java**  
- Entendimento da **pilha de execução (stack trace)**  
- Uso de **try, catch e finally**  
- Lançamento de exceções com **throw e throws**  
- Criação de **exceções personalizadas**  
- Diferença entre **checked e unchecked exceptions**  
- Tratamento de exceções em **APIs REST com Spring Boot**  
- Manipulação e exploração do objeto de exceção  
- Tratamento de múltiplas exceções  
- Hierarquia de exceções  
- Uso de **Controller Advice** para tratamento global  
- Isolamento de respostas de erro  
- Boas práticas no tratamento de exceções
